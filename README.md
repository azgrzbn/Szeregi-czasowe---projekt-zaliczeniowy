# Szeregi czasowe - projekt zaliczeniowy
Projekt zaliczeniowy z przedmiotu analiza szeregów czasowych 2024/2025. Przedmiotem analizy są dane „milk_ts” z pakietu „timeSeriesDataSets”. Dane są przedstawione w formie szeregu czasowego od stycznia 1962 roku do grudnia 1975 roku. Zawiera on informację o miesięcznej produkcji mleka na krowę (w funtach).
